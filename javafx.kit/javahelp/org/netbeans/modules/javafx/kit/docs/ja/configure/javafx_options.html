「フォントと色オプション」ウィンドウの「構文」タブでは、JavaFX Script プログラミング言語に使用できるさまざまな構文カテゴリをカスタマイズできます。

JavaFX Script 言語の構文カテゴリをカスタマイズする

  1. メインメニューから「ツール」>「オプション」を選択します。
  2. 「フォントと色」カテゴリをクリックします。
  3. 「構文」タブをクリックし、言語コンボボックスから JavaFX を選択します。
  4. カテゴリを選択し、使用するフォント、前景色、背景色、エフェクト、およびエフェクトカラーを設定します。
    「プレビュー」区画に、選択した項目がどのように表示されるか表示されます。
  5. 「了解」をクリックして選択した設定を保存します。
