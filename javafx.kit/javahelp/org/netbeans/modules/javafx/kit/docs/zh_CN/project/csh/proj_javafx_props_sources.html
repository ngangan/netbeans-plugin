“源”面板可让您查看项目文件夹的位置，管理源文件夹并指定源编码。

要打开“JavaFX 项目属性”对话框，请右键单击任意 JavaFX 项目，然后选择“属性”。

可以使用面板右侧的按钮添加和删除类路径元素，以及更改它们的顺序。

一个 JavaFX 项目中可以有多个源根目录，但以下情况除外：

  • 源根目录不能存在于另一个 IDE 项目中。
  • 不能将源根目录添加到同一个项目的另一个编译单元中。例如，在“测试包”下注册的源根目录不能添加到“源包”列表中。
  • 所有源根目录被包装到同一个 JAR 文件中，并且共享相同的类路径。
  • 如果几个项目需要使用同一个源根目录，则应该为该源根目录创建单独的项目，并设置项目之间的编译依赖关系。

您可以使用“源”页面底部的“编码”组合框来指定源编码，以便对应用程序进行本地化。
