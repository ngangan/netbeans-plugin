JavaFX の「プロジェクトプロパティー」ダイアログ:「ドキュメント化」

関連項目 

「ドキュメント化」パネルでは、IDE におけるアプリケーションのクラス用の Javadoc ドキュメントの生成方法を設定できます。JavaFX の「プロジェクトプロパティー」ダイアログを開くには、JavaFX プロジェクトを右クリックし、「プロパティー」を選択します。

デフォルトでは、次のオプションが選択されます。

  • Private メンバーと Package Private メンバーを含める。生成されたドキュメントに、すべてのクラスとメンバーが表示されます。
  • @author。 生成されたドキュメントに @author テキストが含まれます。
  • @version。 生成されたドキュメントに @version テキストが含まれます。
  • 生成された JavaFXdoc のプレビュー。 生成された Javadoc ドキュメントをプレビューできます。

追加の Javadoc オプションも追加できます。

Javadoc ドキュメントの生成の詳細は、次を参照してください。
