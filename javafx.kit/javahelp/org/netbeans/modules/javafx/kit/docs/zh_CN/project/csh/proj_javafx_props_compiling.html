通过“编译”面板，可以为项目设置常规编译选项。要添加任何其他编译器选项，请在“其他编译器选项”文本字段中以空格分隔列表的形式输入它们。要打开“JavaFX 项目属性”对话框，请右键单击任意 JavaFX 项目节点，然后选择“属性”。

“编译”面板包含以下选项：

  • 生成调试信息。如果选中此选项，编译的类将包含在断点处停止并使用调试器逐步执行应用程序所需的信息。
  • 报告使用过时的 API。如果选中此选项，若您的代码使用任何过时的 API，则会在编译输出中通知您。
  • 其他编译器选项。以空格分隔列表的形式输入所需的任何其他编译器选项。

注意：

  • 不要使用 -bootclasspath 等编译器选项在 Java 平台之间进行交叉编译。相反，应注册目标 Java 平台，并为项目设置目标 JDK。
  • 通过在对话框左面板中单击“库”，可以管理项目的类路径。通过单击“打包”，可以设置用于 JAR 文件创建的文件过滤器。
