使用“导航”窗口（JavaFX 项目）

“导航”窗口提供了当前选定文件的简洁视图，并且简化了文件不同部分之间的导航。

可通过选择“窗口”>“导航”(Ctrl-7) 来打开“导航”窗口。缺省情况下，将在 IDE 左下角打开“导航”窗口。要在“导航”窗口中显示源文件，请在源代码编辑器、“项目”窗口、“文件”窗口或“收藏夹”窗口中选择该文件。在显示某个文件后，可以使用“导航”窗口将焦点置于文档中包含的特定节点上。为此，请在“导航”窗口中双击指定的节点，光标将自动重定位到文档中的该节点位置。

选择 JavaFX 文件后，将显示选定源文件成员（构造函数、函数和变量）的字母顺序列表。通过将鼠标光标悬停在成员的节点上，可以为成员显示 Javadoc 文档。您可以右键单击某个节点，然后选择以下命令：

  • 转至源。在源代码编辑器中打开类，并将插入点放在节点所表示的构造函数、函数或变量的开始处。
  • 按名称排序。按字母顺序显示成员。
  • 按源排序。按成员在代码中出现的顺序显示成员。
  • 过滤器。打开一个子菜单，可以在其中为要显示的成员类型选择一些公共过滤器。您可以选择以下过滤器的任意组合：
    • 显示继承的成员。显示来自当前类所扩展的类的成员。
    • 显示字段。显示类的字段。
    • 显示静态成员。显示静态字段和方法。
    • 显示非公共成员。显示未提供公共访问权限的成员。

也可以使用“导航”窗口底部的按钮来激活和禁用过滤器。
