JavaFX プロジェクトの移動、コピー、および名前変更

関連項目 

プロジェクトを作成すると、IDE はそのプロジェクトの大部分のメタデータを、プロジェクトの名前と場所に基づいて設定します。それらの値は、プロジェクト構築スクリプトとプロジェクトプロパティーのほか、プロジェクトフォルダの名前と場所に使用されます。プロジェクトの移動、コピー、名前の変更を安全に行うためには、IDE が提供するコマンドを使用する必要があります。

プロジェクトの移動、コピー、または名前の変更を行う

  • 「プロジェクト」ウィンドウでプロジェクトのノードを右クリックし、次のいずれかのコマンドを選択します。
    • 名前を変更
    • 移動
    • コピー

注

  • プロジェクトを新しい場所へ移動またはコピーするときは、コンピュータ上にその新しいフォルダがすでに存在している必要があります。フォルダを作成するには、「参照」ボタンを使用します。
  • プロジェクトの移動や名前の変更を行うと、プロジェクト間のコンパイル時の依存関係が壊れる場合があります。プロジェクトの依存関係を修正するには、移動するか名前を変更したプロジェクトをほかのプロジェクトのクラスパスからいったん削除し、再び追加します。
