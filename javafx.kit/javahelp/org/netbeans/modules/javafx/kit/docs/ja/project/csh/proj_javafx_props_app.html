JavaFX の「プロジェクトプロパティー」ダイアログ:「アプリケーション」

関連項目 

「アプリケーション」区画では、最終アプリケーションの表示に関するプロパティーを定義できます。

JavaFX の「プロジェクトプロパティー」ダイアログを開くには、JavaFX プロジェクトの主ノードを右クリックし、「プロパティー」を選択します。

このページで使用可能なプロパティーは、次のとおりです。

共通のアプリケーションプロパティー。
  • 名前。アプリケーションの名前。
  • ベンダー。アプリケーションの製造元企業の名前。デフォルト値は、システムユーザー ID です。
アプレット固有のプロパティー。
  • 幅。アプレットの幅をピクセル単位で指定します。デフォルトは 200 です。
  • 高さ。アプレットの高さをピクセル単位で指定します。デフォルトは 200 です。
  • ドラッグ可能アプレット。Web ブラウザからデスクトップへのアプレットのドラッグを可能にするプロパティー。この機能を使用するには、Web ブラウザがドラッグ可能なアプレットをサポートしている必要があります。
Web Start 固有のプロパティー。
  • 自己署名付き JAR。選択すると、アプリケーションはプロジェクトの構築時に生成される証明書を使用して署名されます。この証明書は、どの証明書発行局からも証明されていません。アプリケーションを自己署名することで、アプリケーションではディスクにファイルを書き込んだりネットワークにアクセスすることができるようになります。
  • Pack200 圧縮。選択すると、アプリケーションファイルは大幅に圧縮されて、直接配備できるようになります。これによって、帯域幅を節約し、ダウンロード時間を短縮できます。
