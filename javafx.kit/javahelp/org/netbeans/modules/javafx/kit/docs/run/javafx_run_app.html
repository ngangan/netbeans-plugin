Running a JavaFX Application

See Also

With JavaFX projects, you typically have one project that contains the application main class and several projects containing class libraries. You can run the main project or run any individual project that contains an executable class. There are four different ways you can use to run your JavaFX application from the IDE:

  • as a standalone application
  • using the Java Web Start
  • running in a local web browser
  • running in the bundled Mobile Emulator (available on Microsoft Windows platform only)
See Preparing JavaFX Applications for Deployment for more detailed information.

To run the main project:

  • Choose Run > Run Main Project (F6) in the main menu. By default, the IDE uses the standard execution model and runs the project as a standalone application.

To run an individual project:

  • Right-click the project's node in the Projects window and choose Run Project.
  • If you run a project for which you have not specified a main class, the IDE prompts you for the main class. You can change this setting in the Run panel of the project's Project Properties dialog box. Each project can only contain one main class.
