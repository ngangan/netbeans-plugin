“字体和颜色选项”窗口中的“语法”选项卡可让您自定义可用于 JavaFX Script 编程语言的不同语法类别。

为 JavaFX Script 语言自定义语法类别：

  1. 从主菜单中选择“工具”>“选项”。
  2. 单击“字体和颜色”类别。
  3. 单击“语法”选项卡，然后从语言组合框中选择 JavaFX。
  4. 选择一个类别，然后设置要使用的字体、前景色、背景色、效果以及效果颜色。
    “预览”窗格将显示所选内容的显示情况。
  5. 单击“确定”保存您选择的设置。
