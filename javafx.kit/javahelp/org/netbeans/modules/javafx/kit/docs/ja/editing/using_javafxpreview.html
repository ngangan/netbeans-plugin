JavaFX プレビュー機能の使用

関連項目 

JavaFX のプレビュー機能を使用すると、変更するたびにアプリケーションを構築および実行しなくても JavaFX ソースコードに加えた変更を表示できます。エラーが発生していない場合、現在の JavaFX ソースファイルに加えた変更は、ソースエディタ区画のすぐ上にあるキャンバス区画に表示されます。

  • JavaFX プレビュー機能を有効または無効にするには、ソースエディタのツールバーにある「プレビュー」ボタン をクリックします。
  • アクティブなソースファイルの現在の状態を表示するには、ソースエディタのツールバーにある「プレビューのリセット」ボタン をクリックします。
