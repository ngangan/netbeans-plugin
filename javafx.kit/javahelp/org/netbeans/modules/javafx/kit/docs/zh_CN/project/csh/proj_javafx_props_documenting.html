通过“生成文档”面板，可以配置 IDE 为应用程序类生成 Javadoc 文档的方式。要打开“JavaFX 项目属性”对话框，请右键单击任意 JavaFX 项目，然后选择“属性”。

缺省情况下，会选中以下选项：

  • 包括私有成员和软件包私有成员。所有类和成员都显示在生成的文档中。
  • @author。@author 文本包含在生成的文档中。
  • @version。@version 文本包含在生成的文档中。
  • 预览生成的 JavaFXdoc。允许您预览生成的 Javadoc 文档。

还可以添加其他 Javadoc 选项。
