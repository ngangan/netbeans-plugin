「新規 JavaFX アプリケーション」ウィザード: 名前と場所

関連項目 

この「新規 JavaFX アプリケーション」ウィザードパネルでは、プロジェクトの名前を設定したり、プロジェクトのデフォルトの場所を変更できます。アプリケーションを主プロジェクトにしない決定をすることもできます。また、主ファイルを作成しないようにしたり、デフォルト名または選択した名前のどちらで主ファイルを作成するかを選択することもできます。これが最後のパネルです。設定を終了したら、「完了」をクリックしてプロジェクトを作成します。

このパネルにアクセスするには、「ファイル」>「新規プロジェクト」を選択し、「新規プロジェクト」ウィザードでカテゴリとして「JavaFX」、プロジェクトの種類として「JavaFX アプリケーション」を選択します。

パネルには次の構成オプションが表示されます。

  • プロジェクト名。プロジェクトの名前を指定します。この名前が「プロジェクト」ウィンドウに表示されます。この名前は、プロジェクトフォルダおよびコンテキストパスの作成にも使用されます。
  • プロジェクトの場所。プロジェクトのメタデータとソースコードを保存する場所を指定します。
  • プロジェクトフォルダ。プロジェクトのメタデータとソースコードを保存するフォルダを指定します。フォルダは、プロジェクトの場所とプロジェクト名を連結したものです。
  • 空のプロジェクト。デフォルトのソースファイルを持たないプロジェクトの作成を指定します。
  • ソースから。 「フォルダを追加」ボタンをクリックして、新しいプロジェクトの作成に使用するソースファイルを含むフォルダを指定します。
  • 主プロジェクトとして設定。新しく作成されたプロジェクトを「プロジェクト」ウィンドウ内の主プロジェクトとして設定します。
  • 主ファイルを作成。この JavaFX アプリケーション用に主ファイルを作成するかどうかを示します。このファイルのデフォルト名は変更できます。
