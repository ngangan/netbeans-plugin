通过“打包”面板，可以设置过滤器，以确定将哪些文件包含在 JAR 文件中。要打开“标准 Java SE 项目属性”对话框，请右键单击任意 JavaFX 项目，然后选择“属性”。

注意：

  • 要在 JAR 文件中指定清单文件，请打开项目的 project.properties 文件，然后在 manifest.file 属性中输入该清单文件的名称。指定的文件名必须与项目的 build.xml 文件有关。
  • 要更改 JAR 文件的名称和位置，请打开 project.properties 文件，然后编辑 dist.jar 属性。
  • 要禁止为项目生成 JAR 文件，请打开 build.xml，然后覆盖 jar 目标，并从目标中删除所有依赖关系（如下所示）：
    <target name="jar" />
