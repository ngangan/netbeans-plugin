IDE 为编辑和导航使用 JavaFX Script 编程语言编写的源文件提供了以下功能：

  • 语法突出显示。
  • 错误检测。
  • 缩排和重新设置格式。键入时，IDE 将缩排键入的行。可以设置选定行的格式，或者按 Alt-Shift-F 组合键设置整个文件的格式。
  • 代码完成。在您键入内容或按 Ctrl-空格键之后，会出现一个对话框，显示完成您所要键入的 JavaFX 表达式的可能方式。从列表中进行选择，然后查看当前所选内容的 Javadoc 条目。导航到正确的选项，然后按 Enter 完成此表达式。
  • 代码折叠。可让您隐藏或显示源文件中的代码块。
  • 确定导入语句。生成文件中指定的类所需的导入语句。在源文件中的任意位置单击鼠标右键，然后选择“确定导入语句”(Ctrl-Shift-I)。
  • 转到声明语句。允许光标跳转到项目声明语句。
  • 转到源文件。显示包含选定类定义的源文件。
  • 面板。面板中提供了代码片段，可将其拖放到当前在编辑器中打开的 JavaFX 源文件中。
  • 预览。使用 JavaFX 预览功能，查看您对当前源文件的编辑效果。

在源代码编辑器中打开 JavaFX 源文件：

  • 在“项目”窗口中双击 JavaFX 源文件的节点。

在“导航”窗口中查看 JavaFX 源文件：

  • 在“项目”窗口中选择 JavaFX 源文件的节点，然后，导航窗口会显示此文件的各个不同部分。

启用和禁用预览功能：

  • 单击编辑器工具栏上的“预览”按钮 。

编辑器工具栏还为当前显示的源文件提供了其他可用的编辑器命令按钮。有关这些按钮的说明，请参见编辑器工具栏。
