JavaFX™ Plugin for NetBeans 提供了使用 JavaFX Script 编程语言进行编程的功能。JavaFX Script 编程语言是静态类型的声明性语言。它具有相同的代码构造、重用和封装功能，可以通过使用 Java 技术创建和维护非常大的程序。它可以充分利用 JavaFX SDK Runtime 的 GUI 工具包，并可以让您很容易地创建图形用户界面 (graphical user interface, GUI)。要了解有关 JavaFX Script 编程语言的更多信息，请参见中“参考”部分下的链接。

NetBeans IDE 中的 JavaFX 支持包括以下功能：

JavaFX SDK

JavaFX 软件开发工具包 (Software Development Kit, SDK) 提供了可用来生成富 Internet 应用程序并将其部署到浏览器、桌面和移动设备的命令行工具和技术。JavaFX 1.1 SDK 中包括以下组件：

  • JavaFX Desktop 运行时
  • JavaFX Mobile 运行时（仅适用于 Microsoft Windows）
  • JavaFX Mobile Emulator（仅适用于 Microsoft Windows）
  • JavaFX API 文档
  • 示例

它与 JavaFX Plugin for NetBeans 捆绑在一起，并安装在 NetBeans 安装目录中的 javafx2/javafx-sdk 目录下。它已设置为在 IDE 中创建的 JavaFX 项目的缺省平台。有关更多信息，请参见在 JavaFX 项目中设置目标 JavaFX SDK。

JavaFX 编译器

JavaFX 编译器还与 JavaFX 插件集成在一起，用于编译 JavaFX 源代码。可使用 Java 编译器编译 JavaFX 项目中包含的 Java 源代码。编译结果将存储在一个 JAR 文件中，可以将该文件用于部署或分发操作。有关更多信息，请参见关于生成 JavaFX 应用程序。

高级源代码编辑器

这种可识别语言的编辑器可缩排和完成使用 JavaFX Script 编程语言编写的源代码。该编辑器还可执行代码的语义语法突出显示和实时解析，匹配关键字和括号，标记错误和匹配项，并且具有代码完成功能和代码折叠能力。它具有一个工具栏，其中包含可启用和禁用预览功能的控件。有关更多信息，请参见编辑 JavaFX 源文件。

JavaFX 面板

JavaFX 面板提供了可用于快速生成 JavaFX 应用程序的代码片段。可以将代码片段拖放到当前在源代码编辑器中打开的源文件中。有关更多信息，请参见关于 JavaFX 面板。

预览功能

JavaFX 插件包含一个预览功能，可让您查看您对源代码所做的更改效果，而不必在每次进行更改时生成和运行应用程序。如果没有出现编译错误，并且源文件提供了 UI 组件、形状或框架类，则您在源代码编辑器中所做的更改将会显示在源代码编辑器窗格上方的画布窗格中。有关更多信息，请参见使用 JavaFX 预览功能。

调试器

当前，JavaFX 插件中包含的调试器功能支持对 JavaFX 项目中包含的 JavaFX 和 Java 源代码进行调试。有关更多信息，请参见调试 JavaFX 应用程序。

性能分析器

性能分析器功能仅在项目级别可用，在 JavaFX 项目的源级别不可用。有关更多信息，请参见分析 JavaFX 应用程序性能。

部署软件包

使用“项目属性”对话框可以配置和选择用于运行 JavaFX 应用程序的执行模型。IDE 使用此配置来创建对要在 IDE 外部部署的应用程序进行打包所必需的文件。有关更多信息，请参见准备要部署的 JavaFX 应用程序。
