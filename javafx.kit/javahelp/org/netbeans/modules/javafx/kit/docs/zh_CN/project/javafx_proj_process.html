使用 JavaFX 项目

下表说明了如何使用 JavaFX 项目。

创建并配置项目
  1. 通过在主菜单中选择“文件”>“新建项目”(Ctrl-Shift-N) 来创建 JavaFX 项目，然后选择“JavaFX Script 应用程序”模板。
  2. 执行向导中的其余步骤。
  3. 为项目配置类路径。


生成项目

  • 从主菜单中选择“运行”>“生成主项目”(F11)，或在“项目”窗口中右键单击项目节点，然后选择“生成项目”。

运行项目
  1. 如有必要，为项目设置主类和参数。
  2. 从主菜单中选择“运行”>“运行主项目”，或在“项目”窗口中右键单击项目节点，然后选择“运行项目”。

在 JavaFX 项目中调试 Java 代码
  1. 在代码中设置断点或监视。
  2. 从主菜单中选择“运行”>“调试主项目”，或在“项目”窗口中右键单击项目节点，然后选择“调试项目”。也可以将调试器连接至正在运行的进程。
  3. 逐步执行程序。
  4. 在完成调试后，从主菜单中选择“运行”>“完成调试会话”(Shift-F5)。

分析项目性能
  • 从主菜单中选择“性能分析”>“分析主项目性能”，或在“项目”窗口中右键单击项目节点，然后选择“性能分析”。
