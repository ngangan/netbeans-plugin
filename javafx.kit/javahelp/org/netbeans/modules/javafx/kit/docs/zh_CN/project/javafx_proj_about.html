项目由一组源文件以及用来生成、运行和调试这些源文件的设置组成。在 IDE 中，所有 JavaFX 开发都必须在项目内进行。对于包含较大代码库的应用程序，将应用程序源代码分成若干个项目通常是很有用的。

IDE 将直接在 Apache Ant 基础之上构建其项目基础结构，Apache Ant 是领先的 Java 应用程序生成工具。IDE 将项目的所有相关信息存储在一个 Ant 脚本、一个属性文件和若干个 XML 配置文件中。这意味着您可以在 IDE 外部生成和运行项目，其方式与在 IDE 内部完全相同。

IDE 包括一个专为支持 JavaFX 应用程序而设计的 JavaFX 项目模板。

  • 您不需要深入了解 Ant 即可使用 IDE，因为它提供的 JavaFX 项目模板可根据您在 IDE 中输入的选项自动生成 Ant 脚本和属性。
