JavaFX の「プロジェクトプロパティー」ダイアログ:「ライブラリ」

関連項目 

「ライブラリ」パネルでは、プロジェクトの各種クラスパスにライブラリを登録したり、すべてのソースファイルのコンパイルおよび実行の対象となるターゲット JavaFX SDK プラットフォームを設定したりできます。

JavaFX の「プロジェクトプロパティー」ダイアログを開くには、JavaFX プロジェクトのノードを右クリックし、「プロパティー」を選択します。

JavaFX プラットフォーム。すべてのソースファイルのコンパイルと実行に使用するターゲット JavaFX プラットフォームを指定します。デフォルトでは、IDE が実行されている JavaFX SDK プラットフォームが一覧表示されます。コンボボックスから使用可能な別のプラットフォームを選択できます。「プラットフォームを管理」をクリックしてプラットフォームを追加することもできます。

ライブラリ。使用するクラスパスのリスト。クラスパス要素の追加または削除、および要素の順序変更を実行する場合は、パネルの右側にあるボタンを使用します。次を追加できます。

  • JAR ファイル。システム上にある Java アーカイブファイル。
  • ライブラリ。バイナリファイル、ソースファイル、および Javadoc ファイルのコレクション。
  • プロジェクト。別の IDE プロジェクトの構築出力、ソースファイル、および Javadoc ファイル。

クラスパス上のプロジェクトを構築。選択した場合、「構築」コマンドを使用したときに、プロジェクトが依存するすべてのプロジェクトが構築されます。このチェックボックスはデフォルトでオンになっています。このチェックボックスの選択を一時的に解除して、プロジェクトの構築を高速化できます。
