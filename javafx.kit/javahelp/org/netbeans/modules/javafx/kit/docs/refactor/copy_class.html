Copy a Class

See Also

You can copy a class to the same package or a different package.

To copy a class and change the code that references that class:

  1. In the Projects window or Source editor right-click the class and choose Refactor > Copy from the contextual menu.
  2. In the Copy Class dialog box, select the package from the To Package combo box or type the fully qualified package name.
    If you copy to the same package, type a new name for the class.
  3. To apply the changes to the code, do one of the following:
    • Click the Refactor button.
      The IDE instantly applies all the changes.
    • Click the Preview button.
      The Refactoring window displays all lines of code that will be changed. Review the changes and clear the checkbox for any code that you do not want to change. Click the Do Refactoring button to apply the changes.
  • You can also initiate the copying of a class by dragging the class's node in the Project's window to another package or by copying and pasting the class's node.

To copy a class without doing refactoring:

  1. In the Projects window, manually copy the class to another package.
    You can copy and paste the class, or you can use drag and drop.
  2. In the Copy Class dialog box, select the Copy Without Refactoring checkbox.
  3. Click Next
  • You should always perform a clean build after completing any refactoring commands. To perform a clean build, right-click the project's node in the Projects window and choose Clean and Build Project.
