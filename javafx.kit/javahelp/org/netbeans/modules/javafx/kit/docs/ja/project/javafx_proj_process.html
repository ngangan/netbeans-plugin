JavaFX プロジェクトの作業手順

次の表で JavaFX プロジェクトの操作について説明します。

プロジェクトを作成して構築する
  1. JavaFX プロジェクトを作成するには、メインメニューで「ファイル」>「新規プロジェクト」(Ctrl-Shift-N) を選択し、JavaFX Script アプリケーションテンプレートを選択します。
  2. ウィザードによって表示される手順に従います。
  3. プロジェクトの クラスパスを設定 します。


プロジェクトを構築する。

  • メインメニューから「実行」>「主プロジェクトを実行」(F11) を選択するか、「プロジェクト」ウィンドウでプロジェクトノードを右クリックし、「プロジェクトを構築」を選択します。

プロジェクトを実行する
  1. 必要であれば、プロジェクトに 主クラスと引数を設定 します。
  2. メインメニューから「実行」>「主プロジェクトを実行」を選択するか、「プロジェクト」ウィンドウでプロジェクトのノードを右クリックし、「プロジェクトを実行」を選択します。

JavaFX プロジェクトで Java コードをデバッグする
  1. コードに ブレークポイントまたは ウォッチポイントを設定します。
  2. メインメニューから「実行」>「主プロジェクトをデバッグ」を選択するか、「プロジェクト」ウィンドウでプロジェクトノードを右クリックし、「プロジェクトをデバッグ」を選択します。実行時プロセスにデバッガを接続 することもできます。
  3. プログラムをステップごとに実行します 。
  4. デバッグが完了したらメインメニューから「実行」>「デバッグセッションの完了」(Shift-F5) を選択します。

プロジェクトをプロファイルする
  • メインメニューから「実行」>「主プロジェクトのプロファイル」を選択するか、「プロジェクト」ウィンドウでプロジェクトノードを右クリックし、「プロファイル」を選択します。
