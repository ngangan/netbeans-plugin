JavaFX 项目任务：快速参考

本主题介绍了可以对 JavaFX 项目执行的常见任务。有关更多的详细信息，请单击右列中的链接。

要执行此任务... 请按照下列步骤执行操作
创建新的 JavaFX 项目
  1. 选择“文件”>“新建项目”(Ctrl-Shift-N)。
  2. 在“新建项目”向导的“类别”列表中选择 JavaFX。
  3. 在“项目”列表中，选择 JavaFX Script 应用程序，然后单击“下一步”。
  4. 在“名称和位置”面板中，指定项目的名称和位置。
  5. 选择空项目。
  6. 将此项目设置为主项目，然后创建主类文件（如果需要）。
    有关更多信息，请参见“新建 JavaFX 应用程序”向导：名称和位置
导入现有的 JavaFX 应用程序
  1. 选择“文件”>“新建项目”(Ctrl-Shift-N)。
  2. 在“新建项目”向导的“类别”列表中选择 JavaFX。
  3. 在“项目”列表中，选择“JavaFX Script 应用程序”，然后单击“下一步”。
  4. 在“名称和位置”面板中，指定项目的名称和位置。
  5. 选择“来自源”。
  6. 单击“添加文件夹”，导航并选择包含现有源文件的文件夹。
  7. 将此项目设置为主项目，然后创建主类文件（如果需要）。
    有关更多信息，请参见“新建 JavaFX 应用程序”向导：名称和位置
在项目中添加文件
  1. 在“项目”窗口中，选择主项目节点，然后右键单击该节点。
  2. 选择“新建”，然后选择要添加的文件类型。
  3. 在“新建文件”向导中，指定文件名称、位置和软件包。
  4. 单击“完成”创建此文件。
编辑文件
  1. 在“项目”窗口中，选择项目节点，然后双击该节点以将其打开（如果需要）。
  2. 导航至要编辑的文件，然后双击该文件以在源代码编辑器中将其打开。

有关源代码编辑器的更多信息，请参见编辑 JavaFX 源文件。

设置主项目
  • 右键单击项目节点，然后选择“设置为主项目”。
生成项目
  • 选择“生成”>“生成主项目”(F11)，或者右键单击任何项目节点，然后选择“生成项目”。
运行项目
  • 选择“运行”>“运行主项目”(F6) 或者右键单击任何项目节点，然后选择“运行项目”。
调试项目
  • 选择“运行”>“调试主项目”(Ctrl-F5)，或者右键单击任何项目，然后选择“调试项目”。
    注意：在 JavaFX 项目的调试会话期间，只能看到原始 Java 类型。
分析项目性能
  • 选择“性能分析”>“分析主项目性能”，或右键单击任何项目，然后选择“性能分析”。
    注意：当前，只能在项目级别上进行性能分析。
准备要部署的项目
  1. 右键单击项目根节点，然后选择“属性”。
  2. 在“项目属性”对话框窗口中，选择“运行”类别。
  3. 选择要使用的应用程序执行模型。
  4. 指定任何必需的 JVM 参数，然后单击“确定”。
  5. 在“项目”窗口中，选择“运行”>“运行主项目”(F6) 或右键单击项目根节点，然后选择“运行项目”。
    部署应用程序所需的文件可在 /dist 文件夹中找到。
有关可用的部署选项的更多信息，请参见准备要部署的 JavaFX 应用程序。
