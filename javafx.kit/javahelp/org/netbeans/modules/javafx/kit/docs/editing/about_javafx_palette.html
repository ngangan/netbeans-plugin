About the JavaFX Script Palette

See Also

The JavaFX Script Palette window provides code clips that you can use to create your JavaFX Script applications. By dragging and dropping the provided snippets of code, you minimize the amount of typing you do and you can quickly develop your application. Once you drop the code clip, you can immadiately make any necessary modification to the code snippet.

The Palette opens automatically when you open an existing JavaFX Script source file in the source editor. You can also open the Palette by choosing Windows > Palette from the main menu (or Ctrl+Shift+8).

The Palette window view can be customized to alter the way items are sorted, to show or hide item names, or display different sized icons of the listed items. To customize the Palette window view, right-click anywhere in the Palette and choose the appropriate menu item from the contextual menu.

You can use the Palette Manager to add, remove, and organize Palette window components. To open the Palette Manager window, choose Tools > Palette > JavaFX Script Code Clips, or right-click anywhere in the Palette and choose Palette Manager from the contextual menu.

By default, the JavaFX Script Palette has the following categories:

  • Applications. Code clips for Frame, Application and CustomNode classes to get you started with your application.
  • Components. Code clips for visual components, such as Button, Label, and MenuItem.
  • Actions. Code clips for action, mouse and keyboard events
  • Basic Shapes. Code snippets for basic shapes such as circles, rectangles, and polygons.
  • Transformations. Code snippets that allow you to rotate, scale, and translate coordinates.
  • Colors. Code clips for basic colors.
  • Animation. A set of code snippets for defining animation.

To use a clip in your code:

  1. Open your source code file in the JavaFX Editor.
  2. Select and drag the code clip from the Palette to the location where you want it in the source file.
