About the JavaFX Script Palette

See Also

The JavaFX Script Palette window provides code clips that you can use to create your JavaFX Script applications. By dragging and dropping the provided snippets of code, you minimize the amount of typing you do and you can quickly develop your application. Once you drop the code clip, you can immadiately make any necessary modification to the code snippet.

The Palette also opens automatically when you open an existing JavaFX Script source file in the source editor. You can also open the Palette by choosing Tools > Palette > JavaFX Script Code Clips.

The JavaFX Script Palette has the following categories:

  • Applications. Code clips for Frame, Application and CustomNode classes to get you started with your application.
  • Components. Code clips for visual components, such as Button, Label, and MenuItem.
  • Actions. Code clips for action, mouse and keyboard events
  • Basic Shapes. Code snippets for basic shapes such as circles, rectangles, and polygons.
  • Transformations. Code snippets that allow you to rotate, scale, and translate coordinates.
  • Colors. Code clips for basic colors.
  • Animation. A set of code snippets for defining animation using timeline, keyframe,
