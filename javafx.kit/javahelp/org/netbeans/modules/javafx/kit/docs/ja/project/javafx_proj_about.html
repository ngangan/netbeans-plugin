JavaFX プロジェクトについて

関連項目 

プロジェクトとは、複数のソースファイルと、これらのソースファイルを構築、実行、およびデバッグするためのさまざまな設定を指します。IDE で JavaFX を開発する場合、すべての工程はこのプロジェクト内で実行されます。大規模なコードを含むアプリケーションでは、多くの場合、そのソースコードを複数のプロジェクトに分割すると便利です。

プロジェクトのインフラストラクチャーは、Java アプリケーションの主要構築ツールである Apache Ant 上に直接構築されます。プロジェクトに関するすべての情報は、Ant スクリプト、プロパティーファイル、および一部の XML 構成ファイル中に保存されます。これによって、IDE 内でも IDE 外でも、まったく同じ JavaFX プロジェクトを構築および実行できます。

IDE には、JavaFX アプリケーションをサポートするように設計された JavaFX プロジェクトテンプレートが含まれています。

  • 提供される JavaFX プロジェクトテンプレートでは、ユーザーが IDE で入力したオプションに基づいて自動的に Ant スクリプトおよびプロジェクトが生成されるので、IDE で作業を行うために Ant について深く理解する必要はありません。
