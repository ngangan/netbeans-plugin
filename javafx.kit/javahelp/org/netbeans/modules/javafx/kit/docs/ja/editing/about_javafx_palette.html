JavaFX パレットについて

関連項目

JavaFX の「パレット」ウィンドウは、JavaFX アプリケーションの作成に使用できるコードクリップを提供します。提供されたコードクリップをドラッグ＆ドロップすることで、入力の量を最小化し、アプリケーションをすばやく開発できます。コードクリップをドロップしたあとすぐに、このコードに必要な変更を加えることができます。

「パレット」は、既存の JavaFX ソースファイルをソースエディタで開くと自動的に開きます。また、メインメニューから「ウィンドウ」>「パレット」を選択して (または Ctrl-Shift-8 キーを押して)、「パレット」を開くこともできます。

「パレット」ウィンドウの表示はカスタマイズでき、項目のソート方法を変更したり、項目名の表示/非表示を切り替えたり、リストされた項目のさまざまなサイズのアイコンを表示したりできます。「パレット」ウィンドウの表示をカスタマイズするには、「パレット」ウィンドウ内の任意の場所を右クリックし、コンテキストメニューから該当するメニュー項目を選択します。

パレットマネージャーを使用して、「パレット」ウィンドウのコンポーネントを追加、削除、および整理することができます。「パレットマネージャー」ウィンドウを開くには、「ツール」>「パレット」>「JavaFX Script コードクリップ」の順に選択するか、「パレット」内の任意の場所を右クリックし、コンテキストメニューから「パレットマネージャー」を選択します。

デフォルトでは、JavaFX パレットには、次のカテゴリがあります。

  • アプリケーション。アプリケーションを開始するための Stage、Scene、および CustomNode クラス用のコードクリップ。
  • アクション。アクション、マウス、およびキーボードイベント用のコードクリップ。
  • 基本シェイプ。円、矩形、多角形などの基本シェイプ用のコードクリップ。
  • パス。パスを定義する一連のコードクリップ。
  • 変換。 座標を回転、拡大縮小、および変換できるコードクリップ。
  • 色。基本色のコードクリップ。
  • 勾配。線形および放射線状の勾配を定義するためのコードクリップ。
  • アニメーション。アニメーションを定義するための一連のコードクリップ。
  • メディア。メディア機能を実装するための一連のコードクリップ。
  • Swing コンポーネント。チェックボックス、ボタン、ラベルなどの表示コンポーネント用のコードクリップ。

コードにクリップを使用する

  1. ソースエディタでソースコードファイルを開きます。
  2. コードクリップを、「パレット」からソースファイル内の挿入位置にドラッグします。
