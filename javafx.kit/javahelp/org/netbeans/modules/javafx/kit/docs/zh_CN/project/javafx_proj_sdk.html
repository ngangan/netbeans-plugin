在 JavaFX 项目中设置目标 JavaFX SDK

缺省情况下，新创建的 JavaFX 项目会使用 JavaFX 插件捆绑的 JavaFX SDK。它是用于编译、执行和调试 JavaFX 应用程序并分析该应用程序性能的缺省平台。此平台基于 IDE 安装期间选用的 JavaFX 捆绑库和 Java SE 平台 (JDK)。可以通过选择“帮助”>“关于”，然后单击“详细信息”选项卡来查看您的 IDE 安装所使用的 JDK 版本。（在 MacOS 平台上，选择 "NetBeans" >“关于 NetBeans”）。JDK 版本在 "Java" 字段中列出。

  • 通过在命令行或 IDE-HOME/etc/netbeans.conf 文件中使用 --jdkhome  jdk-home-dir 开关启动 IDE，可以使用不同的 JDK 版本来运行 IDE。有关详细信息，请参见 IDE 启动参数。

为 JavaFX 项目设置缺省平台：

  1. 在“项目”窗口中右键单击项目的根节点，然后选择“属性”。
    另外，在 Windows 平台上，您还可以在“项目”窗口中选择项目，从主菜单中选择“文件”>“项目属性”(JavaFX_project_name)。
  2. 在“项目属性”对话框中，选择左窗格中的“库”节点。
  3. 在“JavaFX 平台”组合框中选择所需的 JavaFX 平台，然后单击“确定”。

    切换 JavaFX 项目的目标平台将会执行以下操作：

    • 提供新目标平台的类，以用于代码完成。
    • 显示目标 SDK 的源代码和 Javadoc 文档（如果可用）。
    • 使用目标 SDK 的可执行文件（javafxc 和 javafxrt）编译和执行应用程序。
    • 根据目标 SDK 的库来编译源代码。

向 IDE 注册其他 JavaFX 平台：

  1. 从 IDE 的主菜单中选择“工具”>“Java 平台”。
  2. 从“Java 平台管理器”中，单击“添加平台”按钮。
  3. 在“添加 Java 平台”向导中，选择“JavaFX 平台”类型，然后单击“下一步”。
  4. 指定要分配给新平台配置的“平台名称”。
  5. 指定包含新 JavaFX 平台所需使用的 JavaFX 库的文件夹。使用“浏览”按钮导航到您要使用的 JavaFX SDK 安装所在的文件夹。如果所选平台不受支持，将会显示一条警告，这种情况下建议不要继续使用此平台。
  6. 单击“完成”。
    新平台将添加到可用 JavaFX 平台的列表中。
  7. 单击“关闭”以关闭“Java 平台管理器”对话框。
