JavaFX アプリケーションの構築について

関連項目 

IDE は、Ant 構築スクリプトを使用して、JavaFX アプリケーションを構築します。IDE は、プロジェクトの「プロパティー」ダイアログで入力されたオプションに基づいて、この構築スクリプトを生成します。

構築プロセスは、次の方法でカスタマイズできます。

  • 「プロジェクトプロパティー」ダイアログでクラスパスや JAR フィルタなどの基本的なオプションを入力する。
  • build.xml に含まれる IDE で生成された Ant ターゲットをオーバーライドするか、新しいターゲットを作成する。
  • project.properties 内のプロパティーを編集し、構築出力ディレクトリやファイルの名前と場所を変更する。

JavaFX プロジェクトには、JavaFX と Java 両方のソースファイルを含めることができます。JavaFX コンパイラを使用して JavaFX ソースファイルをコンパイルし、Java コンパイラを使用して Java ソースファイルをコンパイルします。プロジェクトが正常に構築されると、単一の JAR ファイルが作成されて、コンパイルされた JavaFX および Java 両方のコードが含まれます。
