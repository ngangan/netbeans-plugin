JavaFX アプリケーションの実行

関連項目

JavaFX プロジェクトでは、一般に、アプリケーションの主クラスを含むプロジェクトが 1 つと、クラスライブラリを含むプロジェクトが複数あります。主プログラムを実行することも、実行可能なクラスを含む任意のプロジェクトを個々に実行することもできます。JavaFX アプリケーションを IDE から実行するには次の 4 つの方法があります。

  • スタンドアロンアプリケーションとして実行する
  • Java Web Start を使用する
  • ローカルの Web ブラウザで実行する
  • バンドルされているモバイルエミュレータで実行する (Microsoft Windows プラットフォームでのみ使用可能)
詳細は、「JavaFX アプリケーションを配備するための準備」を参照してください。

主プロジェクトを実行する

  • メインメニューの「実行」>「主プロジェクトを実行」(F6) を選択します。デフォルトでは、IDE は標準実行モデルを使用し、プロジェクトをスタンドアロンアプリケーションとして使用します。

個別プロジェクトを実行する

  • 「プロジェクト」ウィンドウでプロジェクトのノードを右クリックし、「プロジェクトを実行」を選択します。
  • 主クラスを指定していないプロジェクトを実行しようとすると、主クラスを指定するよう求められます。この設定は、そのプロジェクトの「プロジェクトプロパティー」ダイアログの「実行」区画で変更できます。1 つのプロジェクトには、主クラスは 1 つしか存在できません。
