通过“库”面板，可以在项目的不同类路径中注册库，并设置用于编译和运行所有源文件的目标 JavaFX SDK 平台。

要打开“JavaFX 项目属性”对话框，请右键单击任意 JavaFX 项目节点，然后选择“属性”。

JavaFX 平台。编译和运行所有源文件时使用的目标 JavaFX 平台。缺省情况下，会列出当前运行 IDE 的 JavaFX SDK 平台。您可以从组合框中选择其他可用的平台。您还可以通过单击“管理平台”来添加平台。

库。要使用的类路径列表。可以使用面板右侧的按钮添加和删除类路径元素，以及更改它们的顺序。您可以添加以下任意内容：

  • JAR 文件。位于系统中某个位置的 Java 归档文件。
  • 库。二进制文件、源文件和 Javadoc 文件的集合。
  • 项目。另一个 IDE 项目的生成输出、源文件和 Javadoc 文件。

依据类路径生成项目。如果选中此选项，在使用“生成”命令时，将生成该项目所依赖的所有项目。缺省情况下，此复选框处于选中状态。可以临时清除此复选框以加快项目的生成速度。
