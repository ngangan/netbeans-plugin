Rename a Class

See Also

To rename a class and change the code that references that class:

  1. In the Projects window or Source editor right-click the class and choose Refactor > Rename from the contextual menu.
  2. In the Rename Class dialog box, type the New Name.
    Select the Apply Rename on Comments checkbox to apply renaming on all occurrences of that class in comments.
  3. If you click the Refactor button, the IDE instantly applies all changes. If you click the Preview button, the Refactoring window displays all lines of code that will be changed. Review the changes and clear the checkbox of any code that you do not want to change. Click the Do Refactoring button to apply the changes.
