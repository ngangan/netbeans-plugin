此“新建 JavaFX 应用程序”向导面板可让您设置项目的名称，并更改其缺省位置。此外，您还可以决定不将其作为主项目，并且选择不创建主文件，或者使用缺省名称或选择的名称来创建主文件。执行完这些操作后，由于这是最后一个窗格，因此，单击“完成”即可创建项目。

通过选择“文件”>“新建项目”，然后在“新建项目”向导中选择 "JavaFX" 类别和“JavaFX Script 应用程序”项目类型，可以打开此面板。

该面板显示了以下配置选项：

  • 项目名称。指定项目的名称，此名称将显示在“项目”窗口中。此名称还用于创建项目文件夹和上下文路径。
  • 项目位置。指定要存储项目 meta 数据和源代码的位置。
  • 项目文件夹。指定要存储项目 meta 数据和源代码的文件夹。此文件夹是由项目位置和项目名称并置而成。
  • 空项目。指定创建不具有任何缺省源文件的项目。
  • 来自源。通过单击“添加文件夹”按钮指定包含要用来创建新项目的源文件的文件夹。
  • 设置为主项目。将新创建的项目设置为“项目”窗口中的主项目。
  • 创建主文件。指示是否为 JavaFX 应用程序创建主文件。您可以更改此文件的缺省名称。
