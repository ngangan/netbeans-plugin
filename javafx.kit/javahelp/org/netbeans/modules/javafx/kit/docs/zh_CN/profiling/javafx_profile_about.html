性能分析是检查应用程序的过程，目的是查找与内存或性能相关的问题。在分析 JavaFX 应用程序性能时，您可以监视 Java 虚拟机 (Java Virtual Machine, JVM) 并获取有关应用程序性能的数据，其中包括方法计时、对象分配和垃圾回收。通过使用此数据，您可以在代码中查找可优化的潜在区域以提高性能。当前，您只能查看原始 Java 数据类型，不能查看 JavaFX 数据类型。

在分析 JavaFX 应用程序性能时，可以指定分析整个应用程序的性能，或可以指定根方法来限制程序校验的代码并控制分析的类。
