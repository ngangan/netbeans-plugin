「ナビゲータ」ウィンドウの使用 (JavaFX プロジェクト)

関連項目

「ナビゲータ」ウィンドウには現在選択中のファイルが縮小表示され、ファイル内の移動を簡単に行えます。

「ナビゲータ」ウィンドウを開くには、「ウィンドウ」>「ナビゲータ」(Ctrl-7) を選択します。デフォルトでは、「ナビゲータ」ウィンドウは IDE の左下隅に表示されます。「ナビゲータ」ウィンドウにソースファイルを表示するには、ソースエディタ、「プロジェクト」ウィンドウ、「ファイル」ウィンドウ、または「お気に入り」ウィンドウでファイルを選択します。ファイルが表示されたら、「ナビゲータ」ウィンドウを使用して、ドキュメントに含まれる特定のノードにフォーカスできます。これを行うためには、「ナビゲータ」ウィンドウで特定のノードをダブルクリックします。すると、カーソルがドキュメント内のノード位置に自動的に再配置されます。

JavaFX ファイルを選択すると、選択したソースファイルのメンバー (コンストラクタ、関数、および変数) のリストがアルファベット順に表示されます。メンバーのノードにマウスを合わせると、そのメンバーの Javadoc ドキュメントを表示できます。ノードを右クリックし、次のコマンドを選択できます。

  • ソースへ移動。ソースエディタでクラスが開かれ、そのノードが表すコンストラクタ、関数、または変数の先頭に挿入ポイントが置かれます。
  • 名前でソート。メンバーをアルファベット順に表示します。
  • ソースでソート。メンバーを、コード内で使用されている順に表示します。
  • フィルタ。サブメニューが表示され、その種類のメンバーの表示用の一般的フィルタを選択できます。次のフィルタの組み合わせを選択できます。
    • 継承メンバーを表示。現在のクラスが拡張するクラスのメンバーを表示します。
    • フィールドを表示。クラスのフィールドを表示します。
    • Static メンバーを表示。static フィールドとメソッドを表示します。
    • 非 Public メンバーを表示。一般アクセスを提供しないメンバーを表示します。

「ナビゲータ」ウィンドウの下にあるボタンを使用して、フィルタを有効にしたり無効にすることもできます。
