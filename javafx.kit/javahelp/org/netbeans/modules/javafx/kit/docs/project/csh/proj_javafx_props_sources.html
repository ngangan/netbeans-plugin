JavaFX Project Properties Dialog Box: Sources

See Also 

The Sources panel enables you to view the location of the project folder, manage source folders as well as specify a source encoding.

You open the JavaFX Project Properties dialog box by right-clicking any JavaFX project and choosing Properties.

Use the buttons on the right side of the panel to add, remove, and change the order of classpath elements.

You can have multiple source roots in a JavaFX project, with the following exceptions:

  • The source root cannot already exist in another IDE project.
  • The source root cannot already be added to another compilation unit of the same project. For example, a source root that is registered under Test Packages cannot be added to the list of Source Packages.
  • All of the source roots are packaged into the same JAR file and share the same classpath.
  • If you have a source root that needs to be used by several projects, you should create a separate project for the source root and set up compilation dependencies between the projects.

You can use the Encoding combo box at the bottom of the Sources page to specify a source encoding to localize your application.
