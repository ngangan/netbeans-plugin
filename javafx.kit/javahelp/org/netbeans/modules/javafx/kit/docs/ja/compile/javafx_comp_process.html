JavaFX プロジェクトの構築手順

次の表は JavaFX プロジェクトを構築する手順をまとめています。

コンパイル時クラスパスを設定する
  • コンパイル時クラスパスを設定します 。


プロジェクトを構築する
  • プロジェクトを構築します 。

コンパイルエラーを修正する
  • 「出力」ウィンドウ を使用して、エラーを見つけ、訂正します。

プロジェクトの生成物を削除して構築
  • build ディレクトリと dist ディレクトリの内容を削除します。

構築出力を検査
  • プロジェクトの build ディレクトリ (「ファイル」ウィンドウ) にある、コンパイル済みクラスを表示します。
  • プロジェクトの dist ディレクトリ (「ファイル」ウィンドウ) にある、JAR ファイルや WAR ファイルなどの Javadoc ファイルおよび構築ライブラリを表示します。
