在“应用程序”窗格中，您可以为最终应用程序的表示形式定义属性。

可以使用以下方法打开“JavaFX 项目属性”对话框：右键单击 JavaFX 项目主节点，然后选择“属性”。

此页提供了如下属性：

通用应用程序属性。
  • 名称。应用程序的名称。
  • 供应商。开发应用程序的公司名称。缺省值是您的系统用户 ID。
特定于 Applet 的属性。
  • 宽度。指定 applet 的宽度，单位为像素。缺省值为 200。
  • 高度。applet 的高度，单位为像素。缺省值为 200。
  • 可拖动的 Applet。可让 applet 从 Web 浏览器拖动到桌面的属性。您的 Web 浏览器必需支持可拖动的 applet 才能实现此功能。
特定于 Web Start 的属性。
  • 自签名 JAR。如果选定该选项，则使用在生成项目时生成的证书对应用程序进行签名。此证书没有经过任何证书授权机构的认证。通过对应用程序进行自签名，应用程序可以执行如下操作：将文件写入硬盘以及访问网络等。
  • Pack200 压缩。如果选中此选项，则应用程序文件会进行高比例压缩，并可直接进行部署，从而可节省带宽并减少下载时间。
