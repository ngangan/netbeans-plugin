JavaFX アプリケーションのプロファイリング

関連項目

プロファイリングとは、メモリーやパフォーマンスに関連した問題を検出するためにアプリケーションを検査するプロセスです。JavaFX アプリケーションをプロファイリングするときは、Java 仮想マシン (Java Virtual Machine、JVM) を監視して、メソッドのタイミング、オブジェクトの割り当て、ガベージコレクションなど、アプリケーションパフォーマンスに関するデータを取得できます。このデータを使用することにより、コード内でパフォーマンスを向上するために最適化可能な潜在的領域を検出できます。現在表示できるのは、生の Java データ型に限られ、JavaFX データ型は表示できません。

JavaFX アプリケーションのパフォーマンスを分析する場合は、アプリケーション全体のパフォーマンスを分析することを指定するか、計測されるコードを制限するルートメソッドを指定してプロファイリングされるクラスを制御できます。
