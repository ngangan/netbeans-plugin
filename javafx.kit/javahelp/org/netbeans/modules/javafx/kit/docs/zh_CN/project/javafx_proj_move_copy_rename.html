在创建项目时，IDE 将基于项目名称和位置来确定项目的大部分 meta 数据内容。在项目生成脚本、项目属性以及项目文件夹的名称和位置中将使用这些值。要安全地移动、复制以及重命名某个项目，您必须使用 IDE 提供的这些命令。

移动、复制或重命名项目：

  • 在“项目”窗口中右键单击项目节点，然后选择以下某项命令：
    • 重命名
    • 移动
    • 复制

注意：

  • 在将项目移到或复制到新位置时，新文件夹必须已存在于计算机上。您可以使用“浏览”按钮来创建文件夹。
  • 移动和重命名项目时，可能会中断项目之间的编译依赖关系。要修复项目依赖关系，请从任何其他项目的类路径中删除重命名或移动的项目，然后再次添加该项目。
