设置 JavaFX 主类和运行时参数

为 JavaFX 项目设置主类和运行时参数：

  1. 右键单击“项目”窗口中的 JavaFX 项目节点，然后选择“属性”。
    另外，您还可以在“项目”窗口中选择项目，从主菜单中选择“文件”>“项目属性”(JavaFX_project_name)。
  2. 在对话框的“类别”窗格中选择“运行”节点。
  3. 在“主类”字段中键入主类的全限定名称（例如， org.myCompany.myLib.MyLibClass ）。主类必须位于项目中，或者位于项目运行时类路径上的一个 JAR 文件或库中。
    • 如果使用“浏览”按钮来选择项目的主类，则文件选择器只显示项目源目录中的类。如果要指定类路径上某个库中的类，则必须在“主类”字段中键入该类的全限定名称。
  4. 选择您所选的应用程序执行模型。
    • 标准执行。IDE 将项目作为一个独立的应用程序执行。
    • WebStart 执行。使用 Java Web Start 执行项目。
    • 在浏览器中运行。IDE 可让应用程序在 Web 浏览器中运行。
    • 在移动仿真器中运行。使用 JavaFX SDK 附随的移动仿真器执行项目。通过从“设备”组合框中进行选择来指定用于移动执行的设备。此设备可以是一个仿真器，也可以是一台与计算机连接的真实设备。
  5. 对于标准执行模型，请在“参数”字段中提供所需的任何应用程序用户参数。
  6. 在“JVM 参数”字段中输入所需的任何运行时参数。

    IDE 将设置项目的主类并存储新添加的参数。

  • 对于已指定主类的 JavaFX 项目，IDE 会自动将项目类路径中的任何 JAR 文件复制到 dist/lib 文件夹中。IDE 还会将每个 JAR 文件添加到应用程序 JAR 的 manifest.mf 文件的 Class-Path 元素中。这会简化在 IDE 外部运行应用程序的过程。
