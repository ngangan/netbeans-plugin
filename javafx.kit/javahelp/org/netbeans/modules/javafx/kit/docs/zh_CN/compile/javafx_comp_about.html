IDE 使用 Ant 生成脚本来生成 JavaFX 应用程序。它将根据您在项目的“属性”对话框中输入的选项来创建生成脚本。

可以通过执行以下任一操作来定制生成进程：

  • 在“项目属性”对话框中输入基本选项，如类路径设置和 JAR 过滤器。
  • 覆盖 IDE 生成的 Ant 目标或在 build.xml 中创建新目标。
  • 在 project.properties 中对属性进行编辑，以更改生成输出目录和文件的名称和位置。

JavaFX 项目可以包含 JavaFX 和 Java 源文件。JavaFX 编译器用于编译 JavaFX 源文件，而 Java 编译器用于编译 Java 源文件。在成功生成项目后，会创建一个 JAR 文件，以包含已编译的 JavaFX 和 Java 源代码。
