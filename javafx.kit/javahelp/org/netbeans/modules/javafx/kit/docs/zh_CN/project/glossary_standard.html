标准项目

使用 IDE 生成的 Ant 脚本来编译、运行和调试应用程序的项目。IDE 提供以下标准项目的模板：

  • Java SE：Java 应用程序、Java 类库、基于现有源代码的 Java 项目、Java 桌面应用程序
  • JavaFX：JavaFX 应用程序
  • Web：Web 应用程序、基于现有源代码的 Web 应用程序
  • Java EE：企业应用程序、基于现有源代码的企业应用程序
  • Enterprise JavaBeans 模块：EJB 模块、基于现有源代码的 EJB 模块
  • 企业应用程序客户端：企业应用程序客户端、基于现有源代码的企业应用程序客户端
  • NetBeans 模块：模块项目、模块套件项目、库包装模块项目
