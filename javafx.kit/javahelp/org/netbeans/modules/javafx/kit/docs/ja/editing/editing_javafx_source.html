JavaFX ソースファイルの編集

関連項目

IDE には、JavaFX Script プログラミング言語で作成されたソースファイルを編集およびナビゲートする次の機能が用意されています。

  • 構文の強調表示。
  • エラー検出。
  • インデントと再フォーマット。IDE は、入力を受け付けながら、行をインデントします｡Alt-Shift-F キーを押すことによって選択した行またはファイル全体をフォーマットできます。
  • コード補完。コードの入力時、または Ctrl-スペースキーを押すと、ダイアログが表示され、入力している JavaFX 式を補完する方法が表示されます。リストから選択し、その現在の選択項目の Javadoc エントリを表示します。正しい選択項目に移動し、Enter キーを押して式を完成します。
  • コード折り畳み。ソースファイルのコードブロックの表示と非表示を切り替えることができます。
  • インポートを修正。ファイル内で指定されたクラスで必要なインポート文を生成します。ソースファイルの任意の場所を右クリックし、「インポートを修正」(Ctrl-Shift-I) を選択します。
  • 宣言へ移動。カーソルの下にある項目の宣言にジャンプできます。
  • ソースへ移動。選択されたクラスの定義を含むソースファイルを表示します。
  • パレット。「パレット」には、現在エディタで開かれている JavaFX ソースファイルにドラッグ＆ドロップできるコードクリップが示されます。
  • プレビュー。JavaFX プレビュー機能を使用して、現在のソースファイルに対して行った編集の効果を表示します。

JavaFX ソースファイルをソースエディタで開く

  • 「プロジェクト」ウィンドウで JavaFX ソースファイルのノードをダブルクリックします。

JavaFX ソースファイルを「ナビゲータ」ウィンドウで表示する

  • 「プロジェクト」ウィンドウで JavaFX ソースファイルのノードを選択すると、ナビゲータにファイルのさまざまな部分が表示されます。

プレビュー機能を有効または無効にする

  • エディタのツールバーにある「プレビュー」ボタン をクリックします。

エディタツールバーには、現在表示されているソースファイルで使用可能なほかのエディタコマンド用のボタンもあります。これらのボタンの説明は、「エディタツールバー」を参照してください。
