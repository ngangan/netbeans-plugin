有了 JavaFX 项目，通常就有了一个包含应用程序主类的项目和几个包含类库的项目。可以运行主项目，也可以运行包含可执行类的任何个别项目。可以使用四种不同的方法从 IDE 中运行 JavaFX 应用程序：

  • 作为独立应用程序
  • 使用 Java Web Start
  • 在本地的 Web 浏览器中运行
  • 在捆绑的移动仿真器中运行（仅可用于 Microsoft Windows 平台）
有关更多详细信息，请参见准备要部署的 JavaFX 应用程序。

运行主项目：

  • 在主菜单中选择“运行”>“运行主项目”(F6)。缺省情况下，IDE 使用标准执行模型，并将项目作为一个独立应用程序运行。

运行个别项目：

  • 在“项目”窗口中右键单击项目节点，然后选择“运行项目”。
  • 如果没有为运行的项目指定主类，IDE 将提示您输入主类。可以在项目的“项目属性”对话框的“运行”面板中更改此设置。每个项目只能包含一个主类。
