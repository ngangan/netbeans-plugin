標準プロジェクト

アプリケーションのコンパイル、実行、デバッグに IDE が生成した Ant スクリプトを使用するプロジェクトです。IDE では、次の標準プロジェクトテンプレートが提供されています。

  • Java SE: Java アプリケーション、Java クラスライブラリ、既存のソースを使用する Java プロジェクト、コンテキスト
  • JavaFX: JavaFX アプリケーション
  • Web: Web アプリケーション、既存のソースを使用する Web アプリケーション
  • Java EE: エンタープライズアプリケーション、既存のソースを使用するエンタープライズアプリケーション
  • エンタープライズ JavaBean モジュール: EJB モジュール、既存のソースを使用する EJB モジュール
  • エンタープライズアプリケーションクライアント: エンタープライズアプリケーションクライアント、既存のソースを使用するエンタープライズアプリケーションクライアント
  • NetBeans モジュール: モジュールプロジェクト、モジュールスイートプロジェクト、ライブラリラッパーモジュールプロジェクト
