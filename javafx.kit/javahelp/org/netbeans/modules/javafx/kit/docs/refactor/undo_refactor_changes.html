Undoing Refactoring Changes

See Also

You can undo any changes that you made during refactoring. When you undo a refactoring operation, the IDE rolls back all changes made to all files that were affected by the refactoring. However, if any of the affected files have been modified since refactoring took place, then the Undo operation is not available.

To undo a refactoring command:

  • Choose Refactor > Undo from the main menu.
