JavaFX プロジェクトの主クラスと実行時引数を設定する

  1. 「プロジェクト」ウィンドウで JavaFX プロジェクトのノードを右クリックし、「プロパティー」を選択します。
    または、「プロジェクト」ウィンドウでプロジェクトを選択し、メインメニューの「ファイル」>「プロジェクトプロパティー」(JavaFX_project_name) を選択します。
  2. ダイアログの「カテゴリ」区画で「実行」ノードを選択します。
  3. 「主クラス」フィールドに主クラスの完全修飾名 (たとえば org.myCompany.myLib.MyLibClass ) を入力します。主クラスは、プロジェクト内か、プロジェクトの実行時クラスパスにある JAR ファイルまたはライブラリのいずれかに存在する必要があります。
    • 「参照」ボタンを使用してプロジェクトの主クラスを選択すると、ファイル選択ダイアログが開き、プロジェクトのソースディレクトリ内のクラスのみ表示されます。クラスパス上にあるライブラリにあるクラスを指定する場合は、「主クラス」フィールドにそのクラスの完全修飾名を入力する必要があります。
  4. アプリケーション実行モデルを選択します。
    • 標準実行。IDE はプロジェクトをスタンドアロンアプリケーションとして実行します。
    • WebStart 実行。プロジェクトは Java Web Start を使用して実行されます。
    • ブラウザで実行。IDE では、アプリケーションを Web ブラウザで実行できます。
    • モバイルエミュレータで実行。プロジェクトは、JavaFX SDK に含まれているモバイルエミュレータを使用して実行されます。「デバイス」コンボボックスからデバイスを選択して、モバイル実行用のデバイスを指定します。デバイスには、エミュレータまたはコンピュータに接続されている実際のデバイスを指定できます。
  5. 標準実行モデルの場合は、「引数」フィールドに必要なアプリケーションのユーザー引数を指定します。
  6. 「JVM 引数」フィールドに必要な実行時引数を入力します。

    プロジェクトの主クラスが設定され、新たに追加された引数が保存されます。

  • 主クラスを指定している JavaFX プロジェクトでは、プロジェクトのクラスパス上の JAR ファイルを dist/lib フォルダに自動的にコピーします。また、IDE は個々の JAR ファイルをアプリケーション JAR の manifest.mf ファイル内の Class-Path 要素にも追加します。これによって、簡単に IDE の外部でアプリケーションを実行できます。
