JavaFX アプリケーションについて

関連項目

JavaFX™ Plugin for NetBeans には、JavaFX Script プログラミング言語でプログラミングを行うための機能が用意されています。JavaFX Script プログラミング言語は、宣言型で、静的に型付けされます。このプログラミング言語には、同じコード構造、再利用、およびカプセル化の機能が備わっているため、Java テクノロジを使用する非常に大きなプログラムを作成および維持できます。この言語では、JavaFX SDK Runtime の GUI Toolkit を集中的に利用して、グラフィカルユーザーインタフェース (graphical user interface、GUI) を簡単に作成できます。JavaFX Script プログラミング言語の詳細は、「」の「Reference」節のリンクを参照してください。

NetBeans IDE の JavaFX サポートには、次の機能が含まれます。

JavaFX SDK

JavaFX Software Development Kit (SDK) には、ブラウザ、デスクトップ、およびモバイルデバイスに対して多彩なインターネットアプリケーションを構築して配備するためのコマンド行ツールとテクノロジが用意されています。JavaFX 1.1 SDK には、次のコンポーネントがあります。

  • JavaFX Desktop runtime
  • JavaFX Mobile runtime (Microsoft Windows のみ)
  • JavaFX Mobile Emulator (Microsoft Windows のみ)
  • JavaFX API ドキュメント
  • サンプル

JavaFX SDK は、JavaFX Plugin for NetBeans にバンドルされており、NetBeans インストールディレクトリの javafx2/javafx-sdk ディレクトリにインストールされます。このサンプルは、IDE で作成された JavaFX プロジェクトを使用するためのデフォルトプラットフォームとして設定されます。詳細は、「JavaFX プロジェクトにおけるターゲット JavaFX SDK の設定」を参照してください。

JavaFX コンパイラ

JavaFX コンパイラも JavaFX Plugin に統合され、JavaFX ソースコードをコンパイルするために使用されます。JavaFX プロジェクトに含まれている Java ソースコードは Java コンパイラでコンパイラされます。コンパイル結果は、1 つの JAR ファイルに格納されます。この JAR ファイルは、配備または配布に使用できます。詳細は、「JavaFX アプリケーションの構築について」を参照してください。

高機能ソースコードエディタ

言語認識エディタによって、JavaFX Script プログラミング言語で作成されたソースコードをインデントし、完成します。このエディタは、意味上の構文の強調表示、コードのライブ解析、単語と括弧の照合、エラーと発生のマーク付けを実行し、コード補完機能とコード折り畳み機能を備えています。エディタには、プレビュー機能を有効または無効にするコントロールを含むツールバーがあります。詳細は、「JavaFX ソースファイルの編集」を参照してください。

JavaFX パレット

JavaFX パレットには、JavaFX アプリケーションをすばやく構築できるコードスニペットが用意されています。コードスニペットは、ソースエディタで現在開いているソースファイルにドラッグ＆ドロップできます。詳細は、「JavaFX パレットについて」を参照してください。

プレビュー機能

JavaFX Plugin は、ソースコードに加えた変更がどのような結果になるか表示できるプレビュー機能を備えており、変更するたびにアプリケーションを構築および実行する必要がありません。コンパイルエラーが発生しておらず、ソースファイルが UI コンポーネント、Shape、または Frame クラスを表している場合、ソースエディタで加えた変更は、ソースエディタ区画のすぐ上にあるキャンバス区画に表示されます。詳細は、「JavaFX プレビュー機能の使用」を参照してください。

デバッガ

現在、JavaFX Plugin に含まれているデバッガ機能では、JavaFX プロジェクトに含まれる JavaFX および Java の両方のソースコードのデバッグをサポートします。詳細は、「JavaFX アプリケーションのデバッグ」を参照してください。

プロファイラ

プロファイラ機能は、プロジェクトレベルでのみ使用でき、JavaFX プロジェクトのソースレベルでは使用できません。詳細は、「JavaFX アプリケーションのプロファイリング」を参照してください。

配備用パッケージ

「プロジェクトプロパティー」ダイアログでは、JavaFX アプリケーションの実行に使用する実行モデルを設定および選択できます。IDE では、この設定を使用して IDE の外部で配備するアプリケーションをパッケージ化するために必要なファイルを作成します。詳細は、「JavaFX アプリケーションを配備するための準備」を参照してください。
