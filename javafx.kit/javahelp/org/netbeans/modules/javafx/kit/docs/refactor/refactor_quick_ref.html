Refactoring: Quick Reference

See Also

Using the IDE's refactoring features, you can change the structure of your code and have the rest of the code modified by the IDE to reflect your changes. For details about the refactoring operation, click the link in the left column.

Refactoring Operation Description
Rename Enables you to change the name of a class, function or variable. The IDE updates source code in your project to reference this element by the new name.
Move Moves a class to a new package. The IDE updates source code in your project to reference this class in its new location.
Copy Copies a class to the same or a different package.
Safely Delete Checks for references to a selected code element and then deletes that element if no references are found.
