JavaFX プロジェクト関連の手順早見表

関連項目

ここでは、JavaFX プロジェクトに対して実行する作業について説明します。詳細な情報を参照するには、右欄に表示されているリンクをクリックしてください。

行う作業... 手順
新規 JavaFX プロジェクトを作成する
  1. 「ファイル」>「新規プロジェクト」(Ctrl-Shift-N) を選択します。
  2. 「新規プロジェクト」ウィザードの「カテゴリ」リストで「JavaFX」を選択します。
  3. 「プロジェクト」リストで、「JavaFX アプリケーション」を選択し、「次へ」をクリックします。
  4. 「名前と場所」パネルで、プロジェクトの名前と場所を指定します。
  5. 「空のプロジェクト」を選択します。
  6. プロジェクトを主プロジェクトとして設定し、必要な場合は主クラスファイルを作成します。
    詳細は、「「新規 JavaFX アプリケーション」ウィザード: 名前と場所」を参照してください。
既存の JavaFX アプリケーションをインポートする
  1. 「ファイル」>「新規プロジェクト」(Ctrl-Shift-N) を選択します。
  2. 「新規プロジェクト」ウィザードの「カテゴリ」リストで「JavaFX」を選択します。
  3. 「プロジェクト」リストで、「JavaFX アプリケーション」を選択し、「次へ」をクリックします。
  4. 「名前と場所」パネルで、プロジェクトの名前と場所を指定します。
  5. 「ソースから」を選択します。
  6. 「フォルダを追加」をクリックして、既存のソースファイルが保存されているフォルダに移動し、選択します。
  7. プロジェクトを主プロジェクトとして設定し、必要な場合は主クラスファイルを作成します。
    詳細は、「「新規 JavaFX アプリケーション」ウィザード: 名前と場所」を参照してください。
プロジェクトにファイルを追加する
  1. 「プロジェクト」ウィンドウで主プロジェクトノードを選択し、右クリックします。
  2. 「新規」を選択して、追加するファイルの種類を選択します。
  3. 「新規ファイル」ウィザードで、ファイル名、場所、およびパッケージを指定します。
  4. 「完了」をクリックしてファイルを作成します。
ファイルを編集する
  1. 「プロジェクト」ウィンドウで、プロジェクトのノードを選択し、必要に応じて、そのノードをダブルクリックして、ノードを開きます。
  2. 編集するファイルがある場所に移動し、ダブルクリックしてファイルをソースエディタで開きます。

ソースエディタの詳細は、「JavaFX ソースファイルの編集」を参照してください。

主プロジェクトを設定する
  • 「プロジェクト」ノードを右クリックし、コンテキストメニューから「主プロジェクトとして設定」を選択します。
プロジェクトを構築する
  • 「構築」>「 主プロジェクトを構築 」(F11) を選択するか、プロジェクトノードを右クリックし、「プロジェクトを構築」を選択します。
プロジェクトを実行する
  • 「実行」>「主プロジェクトを実行」(F6) を選択するか、プロジェクトノードを右クリックし、「プロジェクトを実行」を選択します。
プロジェクトをデバッグする
  • 「実行」>「主プロジェクトをデバッグ」(Ctrl-F5) を選択するか、プロジェクトを右クリックし、「プロジェクトをデバッグ」を選択します。
    注: JavaFX プロジェクトのデバッグセッション中に表示できるのは、生の Java 型のみです。
プロジェクトをプロファイルする
  • 「プロファイル」>「主プロジェクトのプロファイル」を選択するか、任意のプロジェクトをクリックして、「プロファイル」を選択します。
    注: 現在は、プロジェクトレベルでのみプロファイルできます。
配備用にプロジェクトを準備する
  1. プロジェクトのルートノードを右クリックし、「プロパティー」を選択します。
  2. 「プロジェクトプロパティー」ダイアログで「実行」カテゴリを選択します。
  3. 使用するアプリケーション実行モデルを選択します。
  4. 必要な JVM 引数を指定し、「了解」をクリックします。
  5. 「プロジェクト」ウィンドウで、「実行」>「主プロジェクトを実行」(F6) を選択するか、ルートプロジェクトノードを右クリックし、「プロジェクトを実行」を選択します。
    アプリケーションの配備に必要なファイルは、/dist フォルダにあります。
使用可能な配備オプションの詳細は、「JavaFX アプリケーションを配備するための準備」を参照してください。
