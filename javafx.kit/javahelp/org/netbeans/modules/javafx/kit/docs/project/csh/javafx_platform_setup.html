JavaFX Platform Setup Dialog Box

See Also

The JavaFX Platform Setup dialog box enables you to register additional JavaFX platforms within the NetBeans IDE. You open the JavaFX Platform Setup dialog box by selecting JavaFX Platform in the Add Java Platform Wizard and clicking Next.

To register a new JavaFX SDK, do the following:

  • Platform Name. Type the name you want to assign to the new JavaFX platform configuration.
  • JavaFX SDK Folder. Specify the folder that contains the necessary JavaFX libraries to use with the new JavaFX platform. Click the Browse button to navigate to the folder of the JavaFX SDK installation. You will get a warning if the selected platform is not supported. Proceeding to work with this platform is not recommended.
