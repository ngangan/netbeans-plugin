About the JavaFX Palette

See Also

The JavaFX Palette window provides code clips that you can use to create your JavaFX applications. By dragging and dropping the provided code clips, you minimize the amount of typing you do and you can quickly develop your application. Once you drop the code clip, you can immediately make any necessary modification to this code.

The Palette opens automatically when you open an existing JavaFX source file in the source editor. You can also open the Palette by choosing Windows > Palette from the main menu (or Ctrl-Shift-8).

The Palette window view can be customized to alter the way items are sorted, to show or hide item names, or display different sized icons of the listed items. To customize the Palette window view, right-click anywhere in the Palette window and choose the appropriate menu item from the contextual menu.

You can use the Palette Manager to add, remove, and organize Palette window components. To open the Palette Manager window, choose Tools > Palette > JavaFX Script Code Clips, or right-click anywhere in the Palette and choose Palette Manager from the contextual menu.

By default, the JavaFX Palette has the following categories:

  • Applications. Code clips for Stage, Scene and CustomNode classes to get you started with your application.
  • Actions. Code clips for action, mouse and keyboard events
  • Basic Shapes. Code clips for basic shapes such as circles, rectangles, and polygons.
  • Controls. Code clips for UI controls such as buttons, labels, check boxes, and progress bars.
  • Layouts. Code clips for defining layout containers.
  • Paths. A set of code clips for defining a path.
  • Transformations. Code clips that allow you to rotate, scale, and translate coordinates.
  • Colors. Code clips for basic colors.
  • Gradients. Code clips for defining a linear and a radial gradients.
  • Effects. Code clips for defining effects such as Blend, Drop Shadow, and Reflection.
  • Animation. A set of code clips for defining animation.
  • Transitions. Code clips for implementing animated transitions such as Pause Transition, Translate Transition, and Fade Transition.
  • Charts. Code clips for chart controls such as bar chart, pie chart, and line chart.
  • Media. A set of code clips for implementing media features.
  • Swing Components. Code clips for visual components such as CheckBox, Button, and Label.

To use a clip in your code:

  1. Open your source code file in the source editor.
  2. Select and drag the code clip from the Palette to the location where you want it in the source file.
