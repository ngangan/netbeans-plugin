Safely Deleting Visage Code

See Also

Use the Safely Delete refactoring operation to initiate the IDE's check for references to a selected code element before you delete that code element. You can apply Safely Delete on classes, functions, and variables.

To initiate a Safely Delete operation:

  1. Select a code element that you want to delete and choose Refactor > Safely Delete from the main menu
    To select the code element, you can either click the element's node in the Projects window or place a cursor in the element's name in the Source editor.
  2. In the Safely Delete dialog box, click Refactor.
    • If the code element is not referenced by other code, the Safely Delete dialog box is closed and the code element is deleted.
    • If the code element is referenced by other code, the Safely Delete dialog box displays a warning and a Show Usages button.
