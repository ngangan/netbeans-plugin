Visage Project Properties Dialog Box: Documenting

See Also 

The Documenting panel enables you to configure how the IDE generates Javadoc documentation for your application's classes. You open the Visage Project Properties dialog box by right-clicking any Visage project and choosing Properties.

By default, the following options are selected:

  • Include Private and Package Private Members. All classes and members are shown in the generated docs.
  • @ author. The @author text are included in the generated docs.
  • @version. The @version text are included in the generated docs.
  • Preview Generated Visagedoc. Allows you to privew the generated Javadoc documentation.

Additional Javadoc options can also be added.
