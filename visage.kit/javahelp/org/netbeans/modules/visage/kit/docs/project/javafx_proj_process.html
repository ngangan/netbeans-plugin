Working with Visage Projects

The following table shows how to work with Visage projects.

Create and configure project
  1. Create the Visage project by choosing File > New Project (Ctrl-Shift-N) in the main menu and select the Visage Script Application template.
  2. Follow the steps in the remainder of the wizard.
  3. Configure the classpath for the project.


Build project

  • Choose Run > Build Main Project (F11) from the main menu or right-click the project's node in the Projects window and choose Build Project.

Run project
  1. If necessary, set the main class and arguments for the project.
  2. Choose Run > Run Main Project from the main menu or right-click the project's node in the Projects window and choose Run Project.

Debug Java code in a Visage project
  1. Set breakpoints or watches in your code.
  2. Choose Run > Debug Main Project from the main menu or right-click the project's node in the Projects window and choose Debug Project. You can also attach the debugger to a running process.
  3. Step through the program execution.
  4. Choose Run > Finish Debugging Session (Shift-F5) from the main menu when you are finished debugging.

Profile project
  • Choose Profile > Profile Main Project from the main menu or right-click the project node in the Projects window and select Profile.
