Rename a Function or Variable

See Also

To rename a function or variable and update references to that element:

  1. In the Source editor, right-click the function or variable and choose Refactor > Rename from the contextual menu.
  2. In the Rename Field dialog box, type the New Name.
    Select the Apply Rename on Comments checkbox to apply renaming on all occurrences of that element in comments.
  3. To apply the changes to the code, do one of the following:
    • Click the Refactor button.
      The IDE instantly applies all the changes.
    • Click the Preview button.
      The Refactoring window displays all lines of code that will be changed. Review the changes and clear the checkbox for any code that you do not want to change. Click the Do Refactoring button to apply the changes.
