Visage Platform Setup Dialog Box

See Also

The Visage Platform Setup dialog box enables you to register additional Visage platforms within the NetBeans IDE. You open the Visage Platform Setup dialog box by selecting Visage Platform in the Add Java Platform Wizard and clicking Next.

To register a new Visage SDK, do the following:

  • Platform Name. Type the name you want to assign to the new Visage platform configuration.
  • Visage SDK Folder. Specify the folder that contains the necessary Visage libraries to use with the new Visage platform. Click the Browse button to navigate to the folder of the Visage SDK installation. You will get a warning if the selected platform is not supported. Proceeding to work with this platform is not recommended.
