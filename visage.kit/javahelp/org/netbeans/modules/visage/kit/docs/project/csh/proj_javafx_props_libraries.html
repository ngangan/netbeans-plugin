Visage Project Properties Dialog Box: Libraries

See Also 

The Libraries panel enables you to register libraries on a project's various classpaths as well as set the target Visage SDK platform against which all source files are compiled and run.

You open the Visage Project Properties dialog box by right-clicking any Visage project node and choosing Properties.

Visage Platform. The target Visage platform against which all source files are compiled and run. By default, the Visage SDK platform on which the IDE is currently running is listed. You can choose another available platform from the combo box. You can also add a platform by clicking Manage Platforms.

Libraries. The list of classpaths to use. Use the buttons on the right side of the panel to add, remove, and change the order of classpath elements. You can add any of the following:

  • JAR file. A Java archive file somewhere on your system.
  • Library. A collection of binary files, source files, and Javadoc files.
  • Project. The build output, source files, and Javadoc files of another IDE project.

Builld Projects on Classpath. If selected, all projects on which the project depends are built when you use the Build command. This checkbox is selected by default. You can temporarily clear this checkbox to make builds of the project faster.
